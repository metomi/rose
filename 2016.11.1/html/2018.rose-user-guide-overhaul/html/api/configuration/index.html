Rose Configuration¶

Rose is configured by a collection of configuration files which all use the same Rose Configuration Format. They are used for:

Application Configuration
  • rose-app.conf
Suite Configuration
  • rose-suite.conf
  • rose-suite.info
Site And User Configuration
  • rose.conf
Configuration Metadata
  • rose-meta.conf

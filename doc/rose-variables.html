ROSE_DATAC????

Description

The path to the data directory of the cycle time with an offset relative to the current cycle time. ???? is a duration:

  • A __ (double underscore) prefix denotes a cycle time in the future. Otherwise, it is a cycle time in the past.
  • nW denotes n weeks.
  • n or nD denotes n days.
  • Tn or TnH denotes n hours.
  • TnM denotes n minutes.
  • TnS denotes n seconds.

E.g. ROSE_DATACT6H is the data directory of 6 hours before the current cycle time.
E.g. ROSE_DATAC1D and ROSE_DATACT24H are both the data directory of 1 day before the current cycle time.

Provided By

  • rose task-env

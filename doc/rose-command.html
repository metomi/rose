Rose Reference Guide: Command

Sorry, the command reference is under development. In the meantime, you can do the following:

Type rose and rosie on the command line for the command line help.

Type rose help UTIL or rosie help UTIL (where UTIL is a known Rose or Rosie utility as returned by the typing rose or rosie) for the command line help on the specified utility.

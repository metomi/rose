Sphinx Domains
==============

.. _domains: http://www.sphinx-doc.org/en/stable/domains.html

Sphinx includes a system for documenting objects in a structured way via
`domains`_. For example to document a Python function one might do:

.. code-block:: rst

   .. py:function:: answer()

      :return: int The ultimate answer to life, the universe and everything.

      Return 42.

This is what the ``sphinx.ext.autodoc`` system does (e.g. ``autofunction``)
for you automatically.

For documenting Rose components an extension adding a Rose domain has been
added.
